Describe how search engines work

Search engine optimization is the organization of the websites content which uses HTML to improve each page's organic rank. Search engines work by using two different types of that are crawler-based and human-based. Crawler based search engine uses automated systems which "crawl" the web and catalog everything found. The human-based engines require a human to read, review, and rank websites.

Explain the importance of keywords and descriptions as meta tags

Keywords and descriptions are important and used so the search engine can determine how to find the page youre looking for. Keywords are used in the search process and comprised of keywords within the meta tags. Descriptions are a short description of a page's content which will be displayed beneath a listing in the results.
