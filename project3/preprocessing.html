Explain the ways to implement preprocessing:

Some ways to implement preprocessing are using SASS and LESS. The puporse of a preprocessor is a program that takes one type of data and converts it to another type of data.

Define and describes the mechanisms used in either LESS or SASS: mixins, variables, functions, operators, nested variables

The mechanisms used in Sass are variables, for example there are 2 variables that are used mainly are local and global variables. Global variables are able to be accessed from anywhere in the stylesheets. Local variables are declared inside the selector.
